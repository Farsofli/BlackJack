HOW TO INSTALL BLACKJACK GAME:

1. Unzip the python file Blackjack.py
2. Right click and open it with the preferred environment 
(such as Microsoft Visual Studio or your default terminal)
3. You can also open it in the bash terminal, 
change directory (cd) to the Blackjack folder location
4. type open Blackjack.py
5. type Python Blackjack.py
6. Now you can run the game!



HOW TO PLAY THE BLACKJACK GAME:



1. You get the option to choose decks, 
depending on how many decks you choose the game will be longer
2. Betting is done right after, your bank is set to 200 at default
3. if you win, your betting + more will be returned!
